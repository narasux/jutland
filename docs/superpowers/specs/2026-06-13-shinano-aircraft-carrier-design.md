# 信浓号航空母舰添加设计

## 目标

在不重绘用户提供蓝图的前提下，将信浓号以可用的重装甲大型航空母舰加入 Jutland。任务包括拆分并清理俯视图和侧视图、把绿色舰体与舰桥改为统一的偏蓝钢灰色、补全舰船与图鉴配置，并加入 `TestAll` 航母增援和初始舰队。

## 范围

- 使用内部名 `shinano`，展示名 `信浓`，舰种 `aircraft_carrier`，缩写 `CV`。
- 修改 `configs/ships.json5`、`configs/references.json5` 和 `configs/missions.json5`。
- 新增：
  - `resources/images/ships/top/aircraft_carrier/shinano.png`
  - `resources/images/ships/side/aircraft_carrier/shinano.png`
- 保留仓库中与本任务无关的武器 UI、图片和其他未提交改动。
- 不新增武器、弹药、飞机类型或新的 Go 代码。

## 美术处理

### 视图拆分

从仓库根目录的 `shinano.png` 提取侧视图与俯视图。仅做裁剪、透明化、缩放和颜色替换，不修改舰体轮廓、结构线、武器位置或飞行甲板图案。

### 透明背景

先从画布边缘泛洪删除近白背景，再缩放并执行第二次保守泛洪。若索具或栏杆围出的白底仍然存在，先备份当前 PNG，再按连通组件和限定区域清理。中央飞行甲板白线、识别标记和浅灰结构不得作为背景删除。

### 颜色方案

采用用户确认的“加贺风格纯色蓝钢灰”：

- 侧视图舰体、舰桥与绿色舷侧设备统一为低饱和偏蓝钢灰。
- 不保留原绿色迷彩转换形成的深浅几何色块。
- 保留黑色结构线、舷窗、细节轮廓和少量边缘阴影。
- 保留红色船底、灰色飞行甲板、白色甲板线和其他非绿色标记。
- 俯视图仅将绿色舷侧结构转换为同一蓝钢灰色，飞行甲板本身不换色。

正式资源使用确认过的颜色方向，不直接复制临时浏览器预览文件。

### 图片尺寸

资料长度为 266.1 米，配置取整为 `266`。俯视图纵向长度和侧视图宽度目标约为 `1064px`，即 `length * 4`。俯视图保留素材原始视觉比例，不按 36 米型宽强制压缩飞行甲板或舷侧结构。

## 舰船配置

### 基础性能

| 字段 | 值 | 说明 |
| --- | ---: | --- |
| `year` | 1944 | 入役年份 |
| `totalHP` | 71890 | 使用满载排水量 |
| `length` | 266 | 266.1 米取整 |
| `width` | 36 | 36.3 米型宽取整 |
| `maxSpeed` | 27 | 最高航速，节 |
| `horizontalDamageReduction` | 0.55 | 大和级船体与重装甲飞行甲板定位 |
| `verticalDamageReduction` | 0.35 | 体现 95mm 装甲飞行甲板 |
| `acceleration` | 0.22 | 大型重载舰体，低于普通舰队航母 |
| `rotateSpeed` | 0.6 | 低机动性 |
| `fundsCost` | 780 | 高于大凤和现有舰队航母 |
| `timeCost` | 70 | 对应极高生存和战斗持续能力 |

数值以 `taiho`、`kaga` 和 `yamato` 为平衡基准。信浓的核心优势是生存能力和持续运作，不通过高速或超大舰载机数量强化。

### 舰载机

采用用户确认的 1942 年早期计划编制，共 63 架：

- `A6M_0`: 36 架，代表舰载战斗机。
- `B5N2_97`: 18 架，代表舰载攻击机。
- `D3A_99`: 9 架，用现有俯冲轰炸机近似计划中的舰载侦察机，避免新增只服务本舰的飞机类型。
- `takeOffTime`: 2.0 秒，低于轻型航母的出动效率，但保持大型航母可用性。

### 武装

- `JP/127/40/2/DP`: 8 座双联装，左右舷各 4 座。
- `JP/25/60/3`: 35 座三联装，按蓝图可见平台和史实数量分布在两舷。
- 不配置 25mm 单装炮，避免加入大量不可读的细碎挂点。
- 不实现 12 座 120mm 28 联装防空火箭。该武器需要扩展火箭、弹药与战斗配置，超出单舰添加范围，而且信浓出港时并未搭载该火箭系统。

所有武器复用现有配置。挂点位置依据清理后的俯视图设置，射界沿用现有日本航母的左右舷模式。

## 图鉴

`configs/references.json5` 新增同名 `shinano` 条目：

- 类型显示为 `CV / 重装甲航空母舰`。
- 吨位、速度、费用和减伤与舰船配置一致。
- 武装列出 8x2 127mm、35x3 25mm 和 63 架计划航空编制。
- 描述说明其由大和级三号舰改建、未完成即服役，并在首次出航后被美军潜艇射水鱼号击沉。
- 素材作者未知时写 `未知`。
- 链接包含用户提供素材 `shinano.png` 和资料页：
  - https://zh.wikipedia.org/zh-hans/%E4%BF%A1%E6%BF%83%E8%99%9F%E8%88%AA%E7%A9%BA%E6%AF%8D%E8%89%A6

## 验证

完成前执行以下检查：

1. 俯视图和侧视图正式 PNG 均存在，且没有临时备份文件留在资源扫描目录。
2. 两张图片尺寸与 `length * 4` 的目标比例一致，并通过人工预览确认透明背景和纯色钢灰效果。
3. `ships.json5`、`references.json5`、`missions.json5` 和图片文件名统一使用 `shinano`。
4. 所有引用的枪炮和飞机名称在现有配置中存在。
5. 运行 `GOCACHE=/tmp/go-build go build -o /dev/null ./pkg/...`。

根据仓库约定，本任务不主动运行测试。
